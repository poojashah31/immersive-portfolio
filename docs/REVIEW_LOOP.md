# Development Review Loop

For every milestone follow this workflow.

## Step 1

Implement ONLY the requested milestone.

Do not continue ahead.

---

## Step 2

Run the project.

Check

- build errors
- runtime errors
- console warnings

Fix immediately.

---

## Step 3

Review against acceptance criteria.

Ask

Does this feel cinematic?

Is the code modular?

Is the animation smooth?

Is performance acceptable?

Does the implementation match the project vision?

---

## Step 4

If ANY answer is no

Refactor.

Improve.

Run again.

Repeat.

---

## Step 5

Only when every acceptance criterion is satisfied

Mark the milestone complete.

Never continue automatically.

Always wait for confirmation before beginning the next milestone.

After each milestone provide

- Summary
- Files created
- Files modified
- Improvements made
- Remaining issues
- Suggestions (optional)
